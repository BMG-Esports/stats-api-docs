# Legend Object

An object containing a player's most recently reported legend.

## Contents

| Parameter | Type    | Description                                              |
| --------- | ------- | -------------------------------------------------------- |
| name      | String  | The name of the legend.                                  |
| count     | Integer | The number of times the player has reported this legend. |
